# FOS-Agri V3

Static public website demonstration for FOS-Agri.

Live site: https://zakdri.github.io/FOS-agri-V3/

## Current rule

The homepage in `index.html` is approved by the client and must remain visually unchanged. Do not redesign, replace, or rebuild it. Only safe header/menu linking fixes are allowed when needed.

## Phase 1 scope

- Public website
- Static HTML/CSS/JavaScript
- GitHub Pages compatible
- No database
- No backend
- No Hostinger backend dependency

## Development workflow

1. Read `AGENTS.md`, `PROJECT_RULES.md`, and `ARCHITECTURE.md` before editing.
2. Protect the homepage visually.
3. Make small commits.
4. Test the live menu, secondary pages, language switcher, and GitHub Pages compatibility.

# FOS-Agri V3 — No Database Plan

## Phase 1

Phase 1 is static only:

- HTML
- CSS
- JavaScript
- GitHub Pages

No database, backend, authentication, real dashboard, or Hostinger backend dependency should be added in Phase 1.

## Demo-only behavior

If any back office or member area appears in Phase 1, it is a visual/static demonstration only.

## Phase 2

After client approval, Phase 2 may include:

- authentication
- database
- dashboards
- real member space
- real back office
- upload/storage strategy
- hosting decision

These decisions must be made later and documented before implementation.
